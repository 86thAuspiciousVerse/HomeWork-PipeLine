# 课程设计要求：城市公交网络分析与可视化

## 题目

基于高德地图 Web 服务 API，爬取指定城市的公交线路数据，构建公交网络并进行可视化分析。

## 数据来源

- 高德地图 Web 服务 API：获取公交线路名称、线路 ID、站点经纬度、行驶路径等
- 需要自行注册高德开发者账号并获取 API Key

## 要求

1. **数据采集**：通过高德 API 爬取目标城市全部公交线路信息（线路名称、站点名称与坐标、行驶路径坐标）
2. **数据处理**：坐标转换（GCJ-02 → WGS-84）、站点去重
3. **特征分析**：计算平均线路长度、平均站点数、平均站距、非直线系数、环线数量
4. **网络构建**：基于 NetworkX 构建公交换乘网络（Space-P 模型），分析度分布、平均路径长度、关键枢纽站点
5. **可视化**：交互式地图（站点散点 + 线路轨迹）、统计图表，整合为单个 HTML 文件

## 技术栈要求

- Python 3.10+
- pandas、numpy、networkx、Plotly、requests
- 高德地图 Web 服务 API

## 交付物

- 单个自包含 HTML 文件（含交互式地图和统计分析图表）
- 处理后的公交线路数据（CSV）
- 公交网络图数据（GEXF）
